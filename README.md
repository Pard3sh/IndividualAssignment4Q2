# Individual Assignment 4 – Q2: Location Information (Compose + Google Maps)

This app demonstrates location permissions & Google Maps integration.

## Features

1. **Runtime location permission**  

2. **Google Map centered on user location**  

3. **Marker at the user’s location**  

4. **Address display (reverse geocoding)**  

5. **Custom markers on map tap**  

## Key Implementation Details

### Tech stack

- **UI**: Jetpack Compose (Material 3)  
- **Maps**: Google Maps SDK for Android + Maps Compose library  
- **Location**: Fused Location Provider (Play Services Location)
